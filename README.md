# Templates
Configuration templates for Compute Canada projects

**Available templates:**
* RNA-seq (hg19/hg38)
  * Expression
  * Fusion calling
  * Variant calling (gatk/vardict)
* Exome-seq
  * Expression
  * Fusion calling
  * Variant calling (gatk/vardict)
* Copy number variant (CNV) calling
* Whole Genome Bisulfite sequencing (WGBS)
